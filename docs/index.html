简单易用

开发者可以自行编写硬件，最后按照云易连的协议以mdns方式声明自己的端口即可，声明中包含设备相关的基础信息，特别包含设备的型号

相应速度极快

设备可以选择就近的服务器接入设备，并且在一些情况下可以使用p2p的点对点传输，真正让设备的响应速度取决于设备和访问者的网络，基于p2p可以发挥设备和访问者带宽的最大化的利用

设备个性化插件支持

你的智能设备和私有云的控制界面完全由你进行编写，这样可以做出适合开发者的个性化的设备

所有第三方智能音箱支持

云易连可以接收第三方的智能音箱插件以个性化的处理各平台对设备的操作

私有云支持

提供插件可以支持用户自建的私有云存储等，用户使用自有硬盘和带宽托管自己的文件，安全且速度快

支持安卓和iOS等的客户端

开发者界面一次编写多处运行
